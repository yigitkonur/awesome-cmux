# Contributing to Awesome cmux

Thanks for helping improve this list.

## Inclusion criteria

A link should be:

1. Directly relevant to **`manaflow-ai/cmux`**.
2. Useful for users (setup, docs, automation, integrations, discussions, ecosystem, learning).
3. Accessible and stable (or clearly noted if auth/login is required).
4. Described in one concise line.

## Exclusion criteria

- Unrelated `cmux` projects unless listed in **Related Projects** for disambiguation.
- Low-quality AI-generated spam pages with no unique signal.
- Dead links or pages that do not mention/use cmux meaningfully.

## Formatting

Use this format:

```md
- [Name](https://example.com) — Short useful description.
```

Keep sections sorted by relevance (highest signal first).

## Pull request checklist

- [ ] Link opens successfully in a browser.
- [ ] Description is accurate and neutral.
- [ ] Correct section.
- [ ] No duplicate existing entry.
